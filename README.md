# KubeSphere DevOps Agent
builder-maven for kubesphere

## Dependency

- maven: 3.8.4
- openjdk: 11
- git
- docker
- Helm
- kubectl

## How to Build

Enter the directory and execute:

```bash
docker build -f Dockerfile .
```
